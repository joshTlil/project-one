API Tutorial

On this page we will go through the steps of getting and using data from an API. It explains and shows the coding process.

First you need to get the random generated API Key from the website to gain authorize to the sites data

Next you need to go the sites documentation page and see how to utilize the URL and the API Key

Open up your javascript and create a variable that will hold your API Key

Next you will need to make another variable, this variable will hold your URL with the API key and information your that you want to fetch

Now that you have that you need to make a fetch function with the variable URL in the parameter

Next you will create a promise which will be called (.then) and in that promise you will make another function with response in that parameter, next to the parameter you're going to create {}

Inside the {} your are going to make the response return as a .json so it can easily manipulate the data as an array

Now you need to make another promise .then next to the new function with data in the parameter, if you want to see if the data went through you can console.log the data and you can look in the console on you html page

Now that you have the data you will need to make a for loop with the data

Now that you have the data you will want to push the data to the page you want to create two new variables and and element that equal to each other

After you done that you can use the data with the incrament and the data type you want to use

Now you can append the new variables to your html page
